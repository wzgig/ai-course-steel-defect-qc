# 补充材料清单

补充材料为可选加分项，建议尽量准备。

## 建议材料

- 数据集说明：来源、规模、字段、类别、预处理方法。
- 模型权重文件：训练好的权重及加载说明。
- 功能效果截图：界面截图、推理结果截图、测试结果截图。
- 创新点说明：说明大模型能力如何提升传统方案。
- 网站使用指南与案例：说明页面结构、演示顺序、单图检测、批量检测和典型工程案例。
- 参考文献：相关论文、数据集、模型或工具来源。
- 演示视频二维码：如报告附录需要引用视频，可补充二维码。

## 文件命名建议

```text
数据集说明.md
模型权重说明.md
功能效果截图/
创新点说明.md
网站使用指南与案例.md
参考文献.md
```

## 注意事项

- 不要把真实 API 密钥放进补充材料。
- 如果数据集来自公开平台，需要写清楚来源。
- 如果模型权重太大，需要提前确认是否拆分提交。
