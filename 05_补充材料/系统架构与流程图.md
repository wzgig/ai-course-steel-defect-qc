# 系统架构与流程图

## 总体架构

```mermaid
flowchart LR
    A[NEU-CLS 钢材表面缺陷数据集] --> B[数据预处理脚本]
    B --> C[train/val/test CSV 清单]
    C --> D[DefectCNN 模型训练]
    D --> E[best_model.pt 权重]
    E --> F[单图推理模块]
    F --> G[缺陷类别/置信度/推理耗时]
    G --> H[规则知识库报告]
    G --> I[My Codex / OpenAI 兼容 API]
    H --> J[质检解释与处理建议]
    I --> J
    J --> K[Streamlit 质检工作台]
    K --> L[单图检测/批量检测/知识库/案例/指南]
```

## 应用运行流程

```mermaid
sequenceDiagram
    participant U as 用户
    participant UI as Streamlit 页面
    participant M as DefectCNN 模型
    participant R as 质检报告模块
    participant C as My Codex 接口

    U->>UI: 上传图片或选择测试样例
    UI->>M: 调用 predict_image()
    M-->>UI: 返回缺陷类别、置信度、Top-K
    UI->>R: 调用 generate_quality_report()
    alt 已启用外部大模型接口
        R->>C: 发送质检提示词
        C-->>R: 返回自然语言质检报告
    else 未启用或接口不可用
        R-->>R: 使用本地缺陷知识库兜底
    end
    R-->>UI: 返回风险等级、成因、建议和报告
    UI-->>U: 展示检测结果、质检解释和下载入口
```

## 数据处理流程

```mermaid
flowchart TD

    C --> E[验证集 270 张]
```

## 当前页面模块

- 单图检测：上传或选择样例，生成单图质检报告。
- 批量检测：批量推理、统计分布、导出 CSV 与 Markdown 摘要。
- 缺陷知识库：展示六类缺陷的成因和处理建议。
- 模型结果：展示准确率、F1、混淆矩阵和训练曲线。
- 典型案例：提供工程场景演示路径。
- 使用指南：提供网页内操作说明和常见问题。
