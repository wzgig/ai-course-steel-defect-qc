# 网站使用指南与案例

完整操作文档位于：

```text
03_项目代码包/项目名称_团队编号/网站使用指南与案例.md
```

## 快速启动

```powershell
cd 03_项目代码包\项目名称_团队编号
python -m streamlit run app.py
```

访问地址：

```text
http://localhost:8501
```

## 页面说明

| 页面 | 说明 |
| --- | --- |
| 单图检测 | 上传或选择一张图片，查看缺陷分类、置信度、Top-K、风险等级和处理建议 |
| 批量检测 | 对多张图片进行批量推理，导出 CSV 明细和 Markdown 摘要 |
| 缺陷知识库 | 展示六类缺陷的解释、成因和处理建议 |
| 模型结果 | 展示测试准确率、宏平均 F1、混淆矩阵和训练曲线 |
| 典型案例 | 提供生产线抽检、异常批次复核、质检报告辅助三个案例 |
| 使用指南 | 在网页内查看操作步骤和常见问题 |

## 演示案例

| 案例 | 场景 | 演示重点 |
| --- | --- | --- |
| 生产线龟裂抽检 | 产线抽检发现疑似裂纹纹理 | 展示高风险缺陷识别、Top-K 结果和处理建议 |
| 异常批次快速复核 | 同批次多张图片需要快速统计 | 展示批量检测、类别分布、CSV 和摘要下载 |
| 质检报告辅助撰写 | 需要形成质检文字说明 | 展示 My Codex 增强报告和 Markdown 下载 |

## 推荐展示顺序

1. 首页展示测试集准确率、宏平均 F1、测试样本数和接口状态。
2. 单图检测运行一个内置样例。
3. 模型结果页展示混淆矩阵和训练曲线。
4. 批量检测页运行测试集样本并导出结果。
5. 典型案例页说明工程应用价值。
