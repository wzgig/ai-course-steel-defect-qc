## 推荐项目定义

### 项目名称

钢材表面缺陷智能质检与大模型诊断报告系统

### 工程场景

在钢材、带钢、板材等工业生产线中，产品表面可能出现开裂、划痕、夹杂、麻点等缺陷。传统人工质检效率低、主观性强、漏检风险高。本项目使用深度学习识别钢材表面缺陷类别，并通过大模型生成缺陷解释和处理建议，辅助质检人员快速判断问题。

### 输入

- 一张钢材表面缺陷图像。
- 可选：批量图片文件夹。

### 输出

- 缺陷类别。
- 预测置信度。
- 缺陷解释。
- 质量风险等级。
- 处理建议。
- 可导出的质检报告摘要。

### 深度学习模块

- 任务：钢材表面缺陷图像分类。
- 模型建议：ResNet18 或 MobileNetV2。
- 指标：准确率、精确率、召回率、F1 值、单图推理耗时。

### 大模型模块

- 任务：根据模型输出生成中文质检说明。
- 示例能力：
  - 解释缺陷类型含义。
  - 给出可能成因。
  - 给出生产或检修建议。
  - 生成一段报告式结论。

## 数据集收集结果

### 1. NEU-CLS 钢材表面缺陷数据集

状态：已下载到本地，作为当前主推荐数据集。

本地路径：

```text
03_项目代码包/项目名称_团队编号/data/raw/NEU-CLS.zip
03_项目代码包/项目名称_团队编号/data/raw/NEU-CLS/
```

来源：

- Figshare：<https://figshare.com/articles/dataset/NEU-CLS/28903550>
- DOI：<https://doi.org/10.6084/m9.figshare.28903550>

关键信息：

- 数据类型：钢材表面缺陷图像。
- 图像数量：1800 张。
- 类别数量：6 类。
- 每类数量：300 张。
- 图像规格：灰度图，200 x 200 像素。
- 许可：CC BY 4.0。
- 压缩包大小：27,316,154 bytes。
- MD5：`19AF5A1250D8AB0CB0828D31EE2B349D`。

本地类别统计：

| 类别 | 数量 | 中文含义 |
| --- | ---: | --- |
| `crazing` | 300 | 龟裂 |
| `inclusion` | 300 | 夹杂 |
| `patches` | 300 | 斑块 |
| `pitted_surface` | 300 | 麻点表面 |
| `rolled-in_scale` | 300 | 轧入氧化皮 |
| `scratches` | 300 | 划痕 |

注意：

- 当前解压结构包含 `images/` 和 `labels/`，标签为类似 YOLO 的坐标格式。
- 文件名自带类别前缀，可直接构建图像分类任务。
- 原始划分为 train 1770、valid 30。为了报告指标更可信，建议自行按类别重新划分，例如 70% 训练、15% 验证、15% 测试。

### 2. 混凝土裂缝分类数据集

来源：

- Mendeley Concrete Crack Images for Classification：<https://data.mendeley.com/datasets/5y9wdsg2zt/2>

关键信息：

- 总数 40,000 张图像。
- 正负样本各 20,000 张。
- 图像大小 227 x 227，RGB。
- 适合做二分类：有裂缝 / 无裂缝。
- 许可：CC BY 4.0。

适合选题：

```text
混凝土结构裂缝智能检测与维修建议生成系统
```

评价：

- 工程场景明确，尤其适合土木、建筑、道路桥梁方向。
- 二分类任务容易做出高准确率。
- 缺点是题目较常见，大模型模块需要设计得更有价值，例如维修优先级、养护建议、风险说明。

### 3. SDNET2018 混凝土裂缝数据集

来源：

- Utah State University：<https://digitalcommons.usu.edu/all_datasets/48/>

关键信息：

- 超过 56,000 张混凝土裂缝/非裂缝图像。
- 来源包括桥面、墙面、路面。
- 包含阴影、粗糙表面、孔洞、杂物等干扰。
- 数据压缩包约 515 MB。
- 许可：CC BY 4.0。

评价：

- 比普通裂缝二分类更贴近真实结构健康监测。
- 数据较大，下载和训练时间更长。
- 如果最终专业偏土木，这个方向可以替换钢材表面缺陷。

### 4. CWRU 轴承故障诊断数据集

来源：

- Case Western Reserve University Bearing Data Center：<https://engineering.case.edu/bearingdatacenter>

关键信息：

- 提供正常与故障轴承振动数据。
- 实验中记录了不同故障位置、故障尺寸和电机负载条件。
- 适合机械工程、设备运维、预测性维护方向。

适合选题：

```text
电机轴承故障智能诊断与大模型运维建议系统
```

评价：

- 工程含金量高。
- 需要理解振动信号、频域特征、时序模型或一维 CNN，短期实现风险比图像任务高。
- 如果团队中有人熟悉机械振动或信号处理，可以考虑。

### 5. RDD 道路损伤检测数据集

来源：

- RoadDamageDetector 项目：<https://github.com/sekilab/RoadDamageDetector>

关键信息：

- RDD2022 包含多国道路图像，训练集有图像和 Pascal VOC 格式标注。
- 典型类别包括纵向裂缝、横向裂缝、龟裂、坑洞。
- 数据体量较大，检测任务训练成本更高。

适合选题：

```text
道路病害智能识别与养护决策辅助系统
```

评价：

- 工程意义强，报告好写。
- 但目标检测、数据下载、训练和可视化整合都更重。
- 不建议作为当前第一选题，除非直接使用预训练模型或只取小样本演示。

### 6. MVTec AD 工业异常检测数据集

来源：

- MVTec AD：<https://www.mvtec.com/research-teaching/datasets/mvtec-ad>

关键信息：

- 超过 5000 张高分辨率工业检测图像。
- 包含 15 种物体和纹理类别。
- 提供异常像素级标注。
- 许可为 CC BY-NC-SA 4.0，非商业使用。

适合选题：

```text
工业产品异常检测与缺陷解释系统
```

评价：

- 数据集质量高，工业场景强。
- 更偏无监督异常检测，课程项目中解释和实现成本高。
- 如果后续想提升项目层次，可以把它作为对比或拓展数据源，不建议作为第一版主线。

### 7. 用电负荷预测数据集

来源：

- UCI Individual Household Electric Power Consumption：<https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption>

关键信息：

- 2,075,259 条分钟级用电记录。
- 9 个特征。
- 时间跨度接近 4 年。
- 任务适合回归和聚类。
- 许可：CC BY 4.0。

适合选题：

```text
建筑用电负荷预测与大模型节能建议系统
```

评价：

- 数据易处理，LSTM/GRU/Transformer 时间序列预测可以体现深度学习。
- 大模型可生成节能建议和用电异常解释。
- 缺点是视觉展示不如图像检测直观，工程专业性需要靠报告包装。

## 当前建议的技术路线

### 第一版最小可交付

```text
NEU-CLS 数据集
-> 自定义 train/val/test 划分
-> ResNet18 或 MobileNetV2 缺陷分类
-> 输出类别、置信度、混淆矩阵、准确率、精确率、召回率、F1
-> 大模型根据类别生成质检解释和处理建议
-> Streamlit 页面上传图片并展示结果
```

### 可选增强

- 使用标签文件在图片上画出缺陷框，提升展示效果。
- 增加批量检测功能。
- 增加报告导出功能。
- 增加缺陷知识库，将 6 类缺陷的成因和建议写成结构化模板，避免大模型胡编。
