# 人工智能基础 B 期末大作业项目文件夹

项目名称：钢材表面缺陷智能质检与大模型诊断报告系统

当前状态：已完成选题调研、数据集整理、模型训练、My Codex 接口接入、Streamlit 网站、主报告初稿/成稿、DOCX/PDF 导出和补充材料整理。本轮已将网站升级为包含单图检测、批量检测、缺陷知识库、模型结果、典型案例和使用指南的质检工作台。

公开 GitHub 仓库保留代码、报告、模型权重、指标图表、运行截图和 6 张轻量演示样例；完整 NEU-CLS 原始数据集和压缩包不纳入仓库。

## 主要目录

```text

```

## 网站启动

```powershell
cd 03_项目代码包\项目名称_团队编号
python -m streamlit run app.py
```

访问：

```text
http://localhost:8501
```

也可以双击：

```text
03_项目代码包\项目名称_团队编号\run.bat
```
